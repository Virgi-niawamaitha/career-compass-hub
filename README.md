## üì¶ Installation

### Prerequisites

- Node.js 18+ or Bun runtime
- npm, yarn, or bun package manager

### Setup Instructions

1. **Clone the repository**
```bash
git clone https://github.com/Virgi-niawamaitha/career-compass-hub.git
cd career-compass-hub
```

2. **Install dependencies**
```bash
npm install
# or
bun install
```

3. **Start development server**
```bash
npm run dev
# or
bun dev
```

The application will be available at `http://localhost:8080`
